1 """!Object structure for describing MPI programs.
2 
3 Do not load this module directly. It is meant to be loaded only by
4 the produtil.run module.
5 
6 This module handles execution of MPI programs, and execution of
7 groups of non-MPI programs through an MPI interface (which requires
8 all sorts of tricks). This module is also the interface to the
9 various produtil.mpi_impl.* modules that generate the shell command to
10 run MPI programs. This module is built on top of the produtil.prog
11 module and uses it to run the MPI-launching program for your local
12 cluster (mpiexec, mpirun, poe, etc.)
13 
14 In addition, this module contains code to simplify adding new MPI
15 implementations to the produtil.mpi_impl subpackage. High-level code,
16 such as the HWRF scripts, use the produtil.run module to generate
17 object trees of MPIRanksBase objects. The produtil.mpi_impl
18 subpackages then implement an mpirunner function that turns those into
19 a produtil.prog.Runner to be directly executed. The MPIRanksBase
20 object, and its subclasses, implement a few utilites to automate that
21 for you:
22 
23 * to_arglist --- converts the MPI ranks to an mpi launcher command
24  as a produtil.prog.Runner, or to an array of strings
25  for a command file.
26 
27 * nranks --- calculates the number of requested MPI ranks
28 
29 * expand_iter --- iterates over groups of identical MPI ranks
30 
31 * check_serial --- tells whether this program is running MPI programs,
32  or running serial programs as if they were MPI
33  (or both, which most MPI implementations don't support)
34 
35 For MPI implementations that require a command file, see the
36 produtil.mpi_impl.mpi_impl_base CMDFGen class to have the
37 produtil.prog module automatically write the command file before
38 executing the program. The produtil.mpi_impl.mpirun_lsf shows an
39 example of how to use it.
40 
41 See the produtil.run module for full documentation."""

98  def to_arglist(self,to_shell=False,expand=False,shell_validate=None,
99  pre=[],before=[],between=[],after=[],post=[],extra={},
100  include_localopts=False):
101  """!This is the underlying implementation of most of the
102  mpi_impl modules, and hence make_runner as well. It converts
103  this group of MPI ranks into a set of arguments suitable for
104  sending to a Runner object or for writing to a command file.
105  This is done by iterating over either all ranks (if
106  expand=True) or groups of repeated ranks (if expand=False),
107  converting their arguments to a list. It prepends an
108  executable, and can insert other arguments in specified
109  locations (given in the pre, before, between, after, and post
110  arguments). It can also use the to_shell argument to convert
111  programs to POSIX sh commands, and it performs simple string
112  interpolation via the "extra" hash.
113 
114  If to_shell=False then the executable and arguments are
115  inserted directly to the output list. Otherwise (when
116  to_shell=True) the to_shell subroutine is called on the
117  MPIRank object to produce a single argument that contains a
118  shell command. That single argument is then used in place of
119  the executable and arguments. Note that may raise
120  NotValidPosixSh (or a subclass thereof) if the command cannot
121  be expressed as a shell command. In addition, if
122  shell_validate is not None, then it is called on each
123  post-conversion shell argument, and the return value is used
124  instead.
125 
126  You can specify additional argument lists to be inserted in
127  certain locations. Each argument in those lists will be
128  processed through the % operator, specifying "extra" as the
129  keyword list with two new keywords added: nworld is the number
130  of ranks in the MPI program, and "n" is the number in the
131  current group of repeated ranks if expand=False (n=1 if
132  expand=True). Those argument lists are: pre, before, between,
133  after and post.
134 
135  @param to_shell If True, convert executable and arguments to
136  a POSIX sh command instead of inserting them directly.
137  @param expand If True, groups of repeated ranks are expanded.
138  @param shell_validate A function to convert each argument to
139  some "shell-acceptable" version.
140  @param pre Inserted before everything else. This is where you
141  would put the "mpiexec" and any global settings.
142  @param before Inserted before each rank (if expand=True) or group
143  (if expand=False)
144  @param between Inserted between each rank (if expand=True) or group
145  (if expand=False)
146  @param after Inserted after each rank (if expand=True) or group (if
147  expand=False)
148  @param post Appended at the end of the list of arguments.
149  @param extra used for string expansion
150  @param include_localopts If True, then self._localopts is appended
151  between the "before" argument and the command. """
152  kw=dict(extra)
153  kw['nworld']=self.nranks()
154  for x in pre: yield x%kw
155  first=True
156  self._localopts=list()
157  for rank,count in self.expand_iter(bool(expand)):
158  assert(isinstance(rank,MPIRanksBase))
159  assert(isinstance(count,int))
160  if not count>0:
161  continue
162  if first:
163  first=False
164  else:
165  for x in between: yield x%kw
166  kw['n']=count
167  for x in before:
168  assert(isinstance(x,str))
169  yield x%kw
170  if include_localopts:
171  for lo in rank.localoptiter():
172  yield lo
173  if to_shell:
174  if shell_validate is not None:
175  yield shell_validate(rank.to_shell())
176  else:
177  yield rank.to_shell()
178  else:
179  for arg in rank.args(): yield arg
180  for x in after: yield x%kw
181  for x in post: yield x%kw
