Detailed Description

An iter-like object that has a peek() method to peek at the next token without consuming it.

Example:

a=peekable("12345")
a.next() # 1
a.next() # 2
a.next() # 3
a.peek() # 4 <--- peeks at next element
a.next() # 4 <--- peek() did not consume 4, so next() returns 4
a.next() # 5
