Reformat the grid id (MASK_GRID value in the configuration file) if it starts with G.

Looks for G<n> where n is a digit 0-999 and zero pads the value. i.e. G7 becomes G007, G13 becomes G013, etc. Args:

Parameters

Returns
reformatted grid id if valid, None if not
