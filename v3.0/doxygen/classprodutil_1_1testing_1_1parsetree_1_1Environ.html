Looks up the given environment variable.

Searches os.environ for the environment variable named in key. Places the result in a String object with expand_string=False (no string expansion). Environment variables set to the null string will have an empty String() returned.

Returns
the value of the environment variable as a String KeyError if the environment variable is not set

Gets the environment variable, returning it as a python string.

Searches for environment variables in os.environ. It will refuse to look up keys with "." or "%" in them. Otherwise, it is identical to os.environ.__getitem__()

Returns
the environment variable's value as a string ValueError if the key contains "." or "%" KeyError if the environment variable is not set. A variable set to a null string is considered to be set, and will not generate a KeyError.
