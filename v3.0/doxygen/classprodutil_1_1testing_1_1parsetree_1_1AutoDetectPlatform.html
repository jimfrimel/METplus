Detailed Description

Uses a list of Platform objects to detect the local platform.

This involves running scripts at parse time. This is the implementation of the "autodetect" command.

Returns the list of Platform objects detected on this machine.

Resolves "detect" via Platform.resolve() on all platforms given to add(). Evaluates the resulting values in a logical context, and constructs a list with all Platform objects whose logical context is True.

the list of detected platforms. This list will be empty if no platforms are detected.
