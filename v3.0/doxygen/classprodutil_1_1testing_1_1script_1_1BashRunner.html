Creates a bash script for the given arguments.

Parameters
parserThe produtil.testing.parse.Parser containing all needed information. This is used to get the list of runnable tasks and builds, the sets of tasks and builds, and all configuration information.
output_fileThe name of the file to write. This file will contain the bash script.
dry_runIf True, the make_runner only logs what is to be done without actually doing it.
setarithOptional: a string recognized by produtil.testing.setarith.arithparse(). This is used to generate the list of Tasks and Builds to run. If no setarith is given, all Tests and Builds with "run" blocks are run.
