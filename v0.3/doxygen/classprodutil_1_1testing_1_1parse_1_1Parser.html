General subscope parser, used for parsing most scopes.

Parses a scope. This includes the innermost (global) scope, and anything within blocks like these:

( ... declarations and arguments ... )
{ ... declarations ... }
(/ ... array elements ... /)

The various arguments control what is allowed within the scope.

Parameters
tokitera peekable iterator that yields tokens
scopesa list of nested scopes (innermost first) surrounding the scope to parse. The scope being parsed is the first element in this array.
endstokens that indicate the end of the scope
parse_betweentokens that separate pieces of the scope, such as declarations, array elements, or arguments.
allow_overwriteare we allowed to replace values that are already set in the scope?
allow_resolveWhen a variable reference is encountered, are we allowed to resolve it within the scope that is being parsed?
allow_runAre "run" statements allowed in this scope?
allow_noneIf True, then lone variable names are allowed. This is intended for use in argument lists.
allow_useAre "use" statements allowed in this scope?
scope_nameThe name of this scope, used for error reporting.
only_scalarsIf True, then only strings and numbers are allowed to be assigned to variables. If False, anything can be assigned.
allow_loadDo we allow "load" statements in this scope?
seen_runHave we already seen a "run" statement within this scope, before parse_subscope() was called? If so, it is an error to have declarations.
