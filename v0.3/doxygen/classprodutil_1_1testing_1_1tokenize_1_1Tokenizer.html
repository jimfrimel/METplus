Tokenizes the specified file, acting as an iterator over Token objects.

Loops over the text of the given file, creating Token objects and yielding them.

Parameters
textThe text to tokenize.
filenameThe file from which the text originates. This may be used for two purposes. The first is error reporting, and the second is "load" statements, which load files relative to the path to the current file.
first_lineThe line number for the first line of the file.
