produtil.mpi_impl.srun.Implementation Class Reference

Detailed Description

Adds SLURM srun support to produtil.run

This module is part of the mpi_impl package -- see produtil.mpi_impl
for details.  This translates produtil.run directives to SLURM srun
commands.

Detects whether the SLURM srun command is available by looking for it in the $PATH.

Also requires the SLURM_NODELIST variable. This is to detect the case where srun is available, but no slurm resources are available.
