Detailed Description

Base class of the parsetree hierarchy.

This class implements default behaviors representative of any parsetree node. The properties and functions should be overridden by subclasses, as appropriate.

Modifies this object's scopes, and that of any subobjects.

Subclasses that can be expressed as non-constant values must implement this function for rescoping. This function changes the stack of defining scopes, defscopes, based on arguments

Parameters
