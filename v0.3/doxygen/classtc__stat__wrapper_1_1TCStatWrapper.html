def tc_stat_wrapper.TCStatWrapper.build_tc_stat (   self,
  tc_stat_output_dir,
  cur_init,
  tc_input_list,
  filter_opts 
)

This is called from extract_tiles_wrapper and from any other wrapper to provide additional filtering WITHOUT the need for a tc_stat MET config file.

Creates the call to MET tool TC-STAT to subset tc-pairs output based on the criteria specified in the feature relative use case parameter/config file.

Args:

Parameters
tc_stat_output_dirThe output directory where filtered results are saved.
cur_initThe initialization time
tc_input_listThe "list" of input data (the files containing the tc pair data to be filtered) in a string
filter_optsThe list of filter options to apply

Returns: None: if no error, then invoke MET tool TC-STAT and subsets tc-pairs data, creating a filter.tcst file.

Raises CalledProcessError
