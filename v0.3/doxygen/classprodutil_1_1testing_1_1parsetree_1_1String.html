Evaluates this string in a logical context.

Uses string_context() to evaluate this String in a string context, which expands the string if needed. Converts to a logical by looking for one of the following values, case insensitively:

  • .true., true, yes, t, y: return True
  • .false., false, no, f, n: return False
  • anything else: raise ValueError
Returns
True or False if the contents of this string are recognized as a logical value ValueError if the contents of this string are not recognized as a logical value.
